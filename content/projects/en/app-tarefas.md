# Task Management App

A complete task management application developed with modern JavaScript stack, including React frontend and Node.js backend.

## Overview

This project was created to demonstrate fullstack skills, including:

- Interface development with React
- RESTful API creation with Node.js and Express
- Relational database management
- Authentication and authorization implementation
- Application deployment and hosting

## Main Features

### For Users

- **Secure Authentication**: Login and registration with JWT
- **Task CRUD**: Create, read, update and delete tasks
- **Organization**: Categories and tags to organize tasks
- **Filters**: Search and filter tasks by status, priority and date
- **Dashboard**: Overall progress view

### Technical

- **RESTful API**: Well-structured endpoints following best practices
- **Validation**: Data validation on both frontend and backend
- **Security**: Password hashing, SQL injection protection, configured CORS
- **Responsiveness**: Interface adaptable to different screen sizes

## Technologies

### Frontend
- React 18, React Router, Axios, Context API, CSS Modules

### Backend
- Node.js, Express, PostgreSQL, JWT, Bcrypt, Joi
