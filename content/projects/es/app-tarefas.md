# App de Gestión de Tareas

Una aplicación completa de gestión de tareas desarrollada con stack JavaScript moderno, incluyendo frontend React y backend Node.js.

## Visión General

Este proyecto fue creado para demostrar habilidades fullstack, incluyendo:

- Desarrollo de interfaces con React
- Creación de APIs RESTful con Node.js y Express
- Gestión de base de datos relacional
- Implementación de autenticación y autorización
- Deploy y hospedaje de aplicaciones

## Funcionalidades Principales

### Para Usuarios

- **Autenticación Segura**: Login y registro con JWT
- **CRUD de Tareas**: Crear, leer, actualizar y eliminar tareas
- **Organización**: Categorías y etiquetas para organizar tareas
- **Filtros**: Buscar y filtrar tareas por estado, prioridad y fecha
- **Dashboard**: Visualización general del progreso

### Técnicas

- **API RESTful**: Endpoints bien estructurados siguiendo buenas prácticas
- **Validación**: Validación de datos tanto en frontend como backend
- **Seguridad**: Hash de contraseñas, protección contra SQL injection, CORS configurado
- **Responsividad**: Interfaz adaptable a diferentes tamaños de pantalla

## Tecnologías

### Frontend
- React 18, React Router, Axios, Context API, CSS Modules

### Backend
- Node.js, Express, PostgreSQL, JWT, Bcrypt, Joi

## Lo que Aprendí

1. **Arquitectura de APIs**: Cómo estructurar endpoints de forma escalable
2. **Seguridad**: Prácticas de seguridad en aplicaciones web
3. **SQL**: Consultas complejas, índices y optimización
4. **Deploy**: Cómo hacer deploy de aplicaciones fullstack
5. **Pruebas**: Importancia de pruebas automatizadas
