# Portafolio Personal

Este es mi sitio de portafolio personal, construido para mostrar mis proyectos y compartir conocimiento a través de artículos de blog.

## Motivación

Crear un portafolio personal es esencial para cualquier desarrollador que desee:

- Mostrar sus habilidades y proyectos
- Compartir conocimiento con la comunidad
- Tener control total sobre su presencia online
- Demostrar capacidad técnica de forma práctica

## Tecnologías Utilizadas

### Next.js 14

Elegí Next.js por su flexibilidad y recursos modernos:

- **App Router**: Nueva estructura de enrutamiento más intuitiva
- **Server Components**: Mejor rendimiento y SEO
- **Static Site Generation**: Páginas extremadamente rápidas

### TypeScript

TypeScript añade seguridad al código y mejora la experiencia de desarrollo con autocompletado y detección de errores.

### Tailwind CSS

Framework CSS utilitario que permite crear interfaces bonitas rápidamente sin salir del HTML.

### Markdown

Uso archivos Markdown para gestionar contenido de blog y proyectos, facilitando la escritura y mantenimiento.

## Funcionalidades

- **Página de Inicio**: Presentación personal con foto y descripción
- **Blog**: Artículos sobre desarrollo y tecnología
- **Proyectos**: Showcase de mis principales trabajos
- **Diseño Responsivo**: Funciona perfectamente en cualquier dispositivo
- **Dark Mode**: Soporte automático para tema oscuro

## Aprendizajes

Este proyecto me enseñó mucho sobre:

1. Generación de sitios estáticos con Next.js
2. Procesamiento de archivos Markdown en aplicaciones React
3. Diseño de interfaces con Tailwind CSS
4. Estructuración de proyectos TypeScript
