# Personal Portfolio

This is my personal portfolio website, built to showcase my projects and share knowledge through blog articles.

## Motivation

Creating a personal portfolio is essential for any developer who wants to:

- Showcase their skills and projects
- Share knowledge with the community
- Have total control over their online presence
- Demonstrate technical capability in a practical way

## Technologies Used

### Next.js 14

I chose Next.js for its flexibility and modern features:

- **App Router**: New more intuitive routing structure
- **Server Components**: Better performance and SEO
- **Static Site Generation**: Extremely fast pages

### TypeScript

TypeScript adds code safety and improves development experience with autocomplete and error detection.

### Tailwind CSS

Utility CSS framework that allows creating beautiful interfaces quickly without leaving HTML.

### Markdown

I use Markdown files to manage blog and project content, making writing and maintenance easier.

## Features

- **Home Page**: Personal presentation with photo and description
- **Blog**: Articles about development and technology
- **Projects**: Showcase of my main work
- **Responsive Design**: Works perfectly on any device
- **Dark Mode**: Automatic support for dark theme

## Learnings

This project taught me a lot about:

1. Static site generation with Next.js
2. Processing Markdown files in React applications
3. Interface design with Tailwind CSS
4. TypeScript project structure
