# Portfólio Pessoal

Este é meu site de portfólio pessoal, construído para mostrar meus projetos e compartilhar conhecimento através de artigos de blog.

## Motivação

Criar um portfólio pessoal é essencial para qualquer desenvolvedor que deseja:

- Mostrar suas habilidades e projetos
- Compartilhar conhecimento com a comunidade
- Ter controle total sobre sua presença online
- Demonstrar capacidade técnica de forma prática

## Tecnologias Utilizadas

### Next.js 14

Escolhi Next.js pela sua flexibilidade e recursos modernos:

- **App Router**: Nova estrutura de roteamento mais intuitiva
- **Server Components**: Melhor performance e SEO
- **Static Site Generation**: Páginas extremamente rápidas

### TypeScript

TypeScript adiciona segurança ao código e melhora a experiência de desenvolvimento com autocompletar e detecção de erros.

### Tailwind CSS

Framework CSS utilitário que permite criar interfaces bonitas rapidamente sem sair do HTML.

### Markdown

Uso arquivos Markdown para gerenciar conteúdo de blog e projetos, facilitando a escrita e manutenção.

## Funcionalidades

- **Página Inicial**: Apresentação pessoal com foto e descrição
- **Blog**: Artigos sobre desenvolvimento e tecnologia
- **Projetos**: Showcase dos meus principais trabalhos
- **Design Responsivo**: Funciona perfeitamente em qualquer dispositivo
- **Dark Mode**: Suporte automático para tema escuro

## Aprendizados

Este projeto me ensinou muito sobre:

1. Geração de sites estáticos com Next.js
2. Processamento de arquivos Markdown em aplicações React
3. Design de interfaces com Tailwind CSS
4. Estruturação de projetos TypeScript

## Como Usar

```bash
# Clone o repositório
git clone https://github.com/vmussato/vmussato.github.io.git

# Instale as dependências
npm install

# Execute em desenvolvimento
npm run dev

# Build para produção
npm run build
```
